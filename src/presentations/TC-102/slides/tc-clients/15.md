		## Client specialties

		### Node.js
		The Node.js client can run directly in your browser via browserify. This is how the tools.taskcluster.net tools work.

		### Go and Java
		Both these clients provide compile time guarantees that your data structures are valid. No more editing code, running a job for two hours, only to discover you spelt a property name wrong.

		### Python
		For easy integration with the majority of tools at mozilla.
